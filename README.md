## Current Focus: Reliability Engineering & Post-training 🤖

- **LLM Reliability & Monitoring**: Reframing prompt-level drift as an operational monitoring problem. I study how minimal perturbations trigger **Schema Failure Cliffs**—where interface validity can drop from 83.3% to 29.2% under implicit triggers—and build one-way audit chains for model deployment.
- **High-Performance Post-training**: Treating the data -> train -> eval loop as a precision engineering task. My goal is to move beyond simple SFT to robust preference optimization (RLHF/PPO) that survives real-world drift.
